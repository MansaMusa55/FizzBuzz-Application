What is FizzBuzz?

FizzBuzz is a very simple programming task, used in software developer job interviews, to determine whether the job candidate can actually write code. It was invented by Imran Ghory, and popularized by Jeff Atwood. I have experience working with some great programming languages such as C#, Python, SQL, HTML5, CSS3, BOOTSTRAP, and Javascript.

Instructions:
Enter any 2 numbers of your choice then proceed to pick out a second set of numbers to determine what you would like the multiples to be. Those two multiples will determine what will Fizz, Buzz and what will FizzBuzz.
How to get Fizz:

When the user passes in a number into the Fizz input box the application will cycle through the array range and will Fizz if it is a multiple of n number.

How to get Buzz:
When the user passes in a number into the Buzz input box the application will cycle through the array range and will Buzz if it is a multiple of n number.
How to get FizzBuzz
When the user passes in both a number into the Fizz box and into the Buzz box, the application will cycle through the array range and display FizzBuzz if both are multiples of n number.
